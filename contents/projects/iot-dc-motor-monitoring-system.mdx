How It Works (The Tech Behind It)
To achieve real-time monitoring and automated protection, this system integrates several advanced algorithms and sensors:

High-Precision Sensors: Acts as the "eyes" of the system, monitoring current and voltage parameters to detect anomalies accurately.

Firebase Realtime Database: The core infrastructure that allows the robot/system to sync data with low latency, ensuring users see the performance metrics in real-time.

Automated Protection Logic: Reduces risk and uncertainty by implementing a safety threshold; if the motor exceeds safe limits, the system automatically triggers a cutoff.

Remote Monitoring: Allows for probabilistic data logging where users can determine the exact state of the hardware at any given time via a web interface.

Challenges & Achievements
One of the biggest challenges in this project was integrating hardware components with the cloud infrastructure and ensuring the data processing could run smoothly in real-time. The final result is a highly responsive system capable of protecting the DC motor dynamically with a very low margin of error.
